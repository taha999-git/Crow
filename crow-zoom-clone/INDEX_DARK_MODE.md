## 🌟 Key Information

### Text Colors
**Light Mode:** Dark (#1e293b) = READABLE  
**Dark Mode:** Light (#e8e8e8) = READABLE  
**Both:** Contrast ratio 13.4:1 - 15.2:1 (AAA)

### Toggle Button
**Location:** Settings page header (top right)  
**Icon:** ☀️ (light) / 🌙 (dark)  
**Action:** Click to toggle modes

### Persistence
**Storage:** Browser localStorage  
**Duration:** 30+ days  
**Sync:** Same domain only

### Performance
**Impact:** None (CSS only)  
**Animation:** 0.3 seconds  
**FPS:** 60
