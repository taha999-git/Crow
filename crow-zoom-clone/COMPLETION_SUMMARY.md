## 🎬 HOW IT WORKS

### Process Flow
```
User visits website
    ↓
Check localStorage for 'darkMode'
    ↓
Apply dark mode class if found
    ↓
User sees preferred mode!

User clicks toggle button
    ↓
Toggle .dark-mode class on body
    ↓
CSS variables update colors
    ↓
All colors change smoothly
    ↓
Icon switches (☀️ ↔ 🌙)
    ↓
Save preference to localStorage
    ↓
Preference persists for next visit!
```
