## 🎬 HOW IT WORKS

```
1. User visits website
   ↓
   Check localStorage for 'darkMode' preference
   ↓
   Apply dark mode class if preference found

2. User clicks toggle button
   ↓
   Toggle 'dark-mode' class on <body>
   ↓
   CSS variables update colors
   ↓
   All colors change smoothly (0.3s)
   ↓
   Icon switches (☀️ ↔ 🌙)

3. JavaScript saves preference
   ↓
   localStorage.setItem('darkMode', true/false)
   ↓
   Preference persists across sessions

4. Next visit (next day, next week, next month)
   ↓
   Check localStorage
   ↓
   Apply dark mode if preference was true
   ↓
   User sees same mode as before!
```
